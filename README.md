# 🎭 Narratrix — AI Story Creator

🌐 Live Demo: https://arnavsharma-ind.github.io/Narratrix-AI-Story-Creator/

---

## 🚀 Overview

Narratrix is a frontend-only AI-style interactive storytelling web app that runs entirely in the browser.
It allows users to create characters, generate dynamic storylines, and interact through a simple chat system — all without any backend or API keys.

---

## ✨ Features

### 🎭 Character Creation

* Create custom characters
* Define personality and genre
* Stored locally using browser storage

### 📖 Story Mode

* Generates story narratives (2–3 sentences)
* Includes character dialogue
* Provides 3 branching choices
* Emotion-based storytelling

### 💬 Chat Mode

* Chat with your character
* Personality-driven responses
* Maintains conversation history

### 💾 Offline Support

* Uses LocalStorage
* No internet required after loading
* Works fully on GitHub Pages

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Dark UI)
* Vanilla JavaScript
* LocalStorage (Database replacement)

## ⚙️ How It Works

* Character data is stored using LocalStorage
* Story is generated using template-based logic (simulated AI)
* Choices allow branching story progression
* Chat uses character personality for replies
